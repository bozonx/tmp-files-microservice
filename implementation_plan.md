# План решения проблемы сборки в Cloudflare Workers

В логах сборки Cloudflare Workers обнаружены три основные проблемы:
1. **Отсутствие скрипта `build`**: Cloudflare пытается выполнить команду `pnpm run build`, которой нет в `package.json`.
2. **Несоответствие версии Node.js**: В проекте требуется Node >= 24, но используется 22.16.0.
3. **Блокировка скриптов pnpm 10**: Новая версия pnpm требует явного разрешения на запуск скриптов сборки для зависимостей.

## Шаги

1. **Обновление `package.json`**:
    - Добавление скрипта `build`.
    - Изменение требований к версии Node.js на `>=22.0.0` (совместимо с текущей средой Cloudflare).
    - Добавление конфигурации `pnpm.onlyBuiltDependencies` для разрешения сборки `esbuild`, `workerd` и других.
